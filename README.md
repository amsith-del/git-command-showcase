# Git Command Showcase

This repository demonstrates commonly used Git commands.

## Clone Repository
```bash
git clone https://github.com/amsith-del/git-command-showcase.git
``` 
## Check Status
```bash
git status
```

## Stage Files
```bash
git add .
```

## Commit Changes
```bash
git commit -m "message"
```

## Push Changes
```bash
git push
```

## Pull Latest Changes
```bash
git pull
```

## Create Branch
```bash
git checkout -b branch-name
```

## Git Stash
```bash
git stash
```
